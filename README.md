# Sentinel

**The Intelligent Custodian for Your Digital Workspace.**

Sentinel is a local-first AI agent designed to maintain your computer's health through intelligent file organization and safe cleanup strategies. By leveraging local LLMs, Sentinel understands the context of your files—identifying forgotten installers, misplaced screenshots, and duplicate archives—while ensuring your data never leaves your machine.

## ⚡ Key Capabilities

*   **Intelligent Classification**: Beyond simple extensions; Sentinel understands file content and usage patterns.
*   **AI-Generated Planning**: Cleanup plans are dynamically generated by local LLMs (Ollama) tailored to your habits.
*   **Safety-First Engine**: Multi-layer validation ensures system directories are never touched and all actions are reversible.
*   **Zero-Cloud Dependency**: 100% telemetry-free and local execution. Privacy is a feature, not an afterthought.
*   **Universal Interface**: Seamless experience across CLI, Web, and Native Desktop environments.

## 🚀 Quick Start

### 1. Prerequisites
Ensure you have **Python 3.11+**, **Node.js 18+**, and **Ollama** installed.

```bash
# Pull the recommended model
ollama pull llama2
```

### 2. Installation
```bash
git clone https://github.com/mystic/sentinel.git
cd sentinel/sentinel-core
poetry install
```

### 3. Immediate Scan
```bash
sentinel clean-pc scan ~/Downloads
```

## 🛡️ Safety Guarantees

Sentinel was built on the principle of **Informed Consent**.

1.  **AI Never Executes**: The AI generates a proposal; a deterministic engine performs the action only after user approval.
2.  **Trash-First**: By default, "deletes" move files to the system Trash/Recycle Bin.
3.  **Undo Everything**: Every session generates an operation log that can be fully reverted.
4.  **System Lockdown**: Hardcoded protections for `/System`, `/Windows`, and critical user directories.
